About Project

Class Management System is a complete class management software designed to automate class events. Diverse operations for classes, staffs and events are available.

This class management software has a powerful online community to bring teacher and student information on a common platform. It is an office automation solution for today's modern classes and tutions. The Class Management System provides the facility to carry out all day to day activities of the class or tuition.
